"Iterative Back-Translation for Neural Machine Translation"

sobamchan, 19/01/28, Misc

1. 論文

Iterative Back-Translation for Neural Machine Translation
Proceedings of the 2nd Workshop on Neural Machine Translation and Generation

2. どんなもの？

NMTを学習するためのparallel corpusを自動生成でかさ増しした。

3. 先行研究と比べてどこがすごい？

Back-Translationを使用して、monolingual corpusからparallel corpusを生成した。

4. 技術や手法のキモはどこ？

実際にほしい方向のNMT(model_forward) (source -> target)とは逆方向(target -> source)モデル (model_back)を先に学習し、 そのモデル (model_back)を使用し、target言語のmonolingual corpusを入力としてparallel corpusを生成する。
それを元のparallel corpusに追加して順方向(source -> target)モデル (model_forward)を学習。
次はこのmodel_forwardを使用して、source言語のmonolingual corpusからparallel corpusを生成する。 これを元データに追加して最初に戻ってmodel_backを再学習。
これを収束するまで繰り返す。

5. どうやって有効だと検証した？

大量に元データがある条件と、少ない時両方の場合で実験を行った。 結果として両方の条件で十分機能することを実験的に示した。

6. 議論はある？

Back-Translationにより生成された合成データセットの精度が大きく最終的なシステムの精度に関連することが確認され、 良い合成サンプルを選択して使用するなどの方法が考えられる。
