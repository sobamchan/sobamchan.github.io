"How to check if parameters are updated in pytorch"

sobamchan, 19/01/09, Misc

目的

optimizerをたくさん用意したりして、色々ごちゃごちゃした実装をしたりしていると networkのパラメーターがちゃんと更新されているか、または固定しているパラメーターが ちゃんと固定されているかどうかが不安になるときがたまにある。
きれいな実装にしていればそれで問題ないのがた、短絡的に確認することが結構あるのでここで 簡単な実装だが書いておく。

方法

import copy
# ...

net = Model(args)
o = optim.Adam(net.parameters())

old_param = copy.deepcopy(list(net.parameters()))
# ... netにデータを入力してloss.backward(), o.step()とか色々、、、
new_param = copy.deepcopy(list(net.parameters()))

print('Updated?: ', (not (new_param == old_param).all()))
